Projeto

Como foi feito?

O projeto foi realizar com o intuito de mostrar como é feito o passo-a-passo de uma instalação de rede de computador, demonstrando os custos, vantagens e desvantagens de cada passo. Se você pode sonhar, você pode realizar! Walt Disney

Instalação

A instalação da rede foi efetuada usando 29 computadores, sendo onze para a sala de informática, doze para a sala de multimídia, três para a diretoria e três para a recepção, com a necessidade de instalar um servidor para os computadores, utilizando um IP estático (fixo) de modo que não mude, a fim de facilitar a manutenção e programação dos computadores. Também foi instalado um roteador, localizado no corredor entre as quatro salas, de modo que o sinal seja distribuído de maneira uniforme, com o mesmo tendo um IP dinâmico (DHCP), de modo que não precise configurar o IP manualmente.

Procedimento

O procedimento foi realizado medindo a área da escola, nesse caso 80m², sendo 4 metros de frente por 20 metros de lado. Em seguida, foi utilizado um multímetro para verificar a tensão da rede para não causar danos no computador, com isso, foi constatado que a tensão da rede é bivolt, ou seja, pode trabalhar tanto em 127V quanto em 220V. A ordem de serviço foi expedida pela diretoria da escola e assinada posteriormente pela equipe contratada, no caso, a equipe Alpha: Mateus Schiavi, Matheus Celestino e Bruno Soares. Após isso, o resto do procedimento foi realizado junto com a instalação, sendo que todos os computadores, com exceção da impressora e do computador (PC-0) localizado na diretoria, foram configurados com IP dinâmico (DHCP), de modo que a segurança aumente e cada vez que o sistema for reiniciado, o IP mude, a fim de atenuar a probabilidade de invasão e vazamento de dados.

Posso instalar uma rede de computadores?

Sim, você pode instalar uma rede de computadores desde que você seja habilitado para isso. Há muitos riscos na hora de instalar uma rede, como perda de arquivos, mal funcionamento e danos irreversíveis. Caso você não saiba como fazer, procure um técnico em TI ou alguém que tenha licença para isso.

Vantagens e desvantagens de IP estático (fixo) e IP dinâmico(DHCP)

Vantagem de um IP estático (fixo): maior facilidade na hora de localizar e agilidade na hora da manutenção. Desvantagem de um IP estático (fixo): mais sensível a ataques de hacker. Vantagem de um IP dinâmico (DHCP): maior segurança e mais agilidade na hora de programar a rede. Desvantagem de um IP dinâmico (DHCP): por ser um IP dinâmico, ou seja, variável, é mais difícil de localizar o computador na hora da manutenção.
